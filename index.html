Waarom laad een font niet in?

  • Browser blokkeert font
  • Langzaam internet
  • Fontformaat word niet gesupport
  • Fout in jouw css

Wat gebeurt er als jouw font niet in laad?

Standaard font word laten zien, (De Flash of unstyled techniek)

gebeurt bij de browsers:

  1. Chrome (Blink-engine)
  2. Firefox (Gecko-engine)
  3. Edge (moderne versie, gebaseerd op Chromium)
  4. Opera

Er word geen tekst laten zien tot het font word in geladen

gebeurt bij de browsers:

  1. Safari
  2. Oude versies van Edge

Hoe kan je het gedrag aan passen?

  • Gebruik de font-display property
  • Gebruik de font-swap property

Werkt het of werkt het niet?

    Als een custom font niet geladen kan worden, gebruikt de browser een standaard fallback font. Dit kan verschillen per besturingssysteem en browser.
  • Serif: Times new roman
  • Sans-serif: Arial, Helvetica
  • Monospace: Courier
